# Puzzle Escape

## Overview

Puzzle Escape is a 2D grid-based puzzle game developed in Unity as part of a game development internship.

Players control a party of three unique characters and must solve puzzles by interacting with buttons, doors, keys, and other mechanics while using as few Action Points as possible.

The objective of every level is to guide all three party members to the exit.

---

## Features

- Grid-based movement
- Three-character party system
- Action Point system
- Buttons and doors
- Keys and locked doors
- Level progression
- Level restart system
- Modular architecture for adding new mechanics
